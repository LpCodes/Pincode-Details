
# Indian Postal PIN Code Details

A simple package to fetch PINCODE Details

## Usage/Examples

```
from IndianPinCodes import getpin

getpin(110001)

Info for Pincode 110001 is as follows 

Name --> Supreme Court

Region --> Delhi

District --> Central Delhi

Division --> New Delhi Central

Region --> Delhi

Block --> New Delhi

Circle --> Delhi

State --> Delhi

```
